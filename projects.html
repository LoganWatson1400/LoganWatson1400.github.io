Emotion Classification

A comparative CNN study that benchmarks three architectures against a 6-class facial emotion dataset. The project trains a baseline CNN (no pooling), a regularized CNN with MaxPooling and 0.5 Dropout, and a fine-tuned Xception transfer learning model Results are logged per-step and exported to a structured table, with loss and accuracy curves plotted for direct model comparison. The fine-tuned Xception model achieved the best validation accuracy at ~33%, demonstrating diminishing returns of transfer learning on a small dataset without full unfreezing.

Discord Git Push-Viewer

A Node.js bot that monitors a GitHub repository for pull request activity and sends real-time Discord notifications. Posts an embed when a PR is opened, then replaces it with a merge or close notification when the PR is resolved. State is persisted with Upstash Redis so no PR gets double-notified across restarts.

Schedule Optimizer

Production schedule optimizer for Kimberly-Clark manufacturing lines, using a genetic algorithm to minimize downtime and meet demand across tissue machines and converting lines.

RISA MIPS Assembler

A two-milestone MIPS assembler written in Java that translates Reduced Instruction Set Architecture (RISA) assembly into 8-digit hexadecimal machine code. Milestone 1 handles single-line assembly parsing via a JAR CLI tool; Milestone 2 extends it into a full MIPS simulator capable of processing complete .asm programs with separate .text and .data segments.

3D Pong

A 3D Pong game built from scratch in C++ using OpenGL and GLUT. The player controls a rectangular paddle in 3D space to deflect a bouncing ball, which accelerates with every successful hit. Features include real-time shadow projection onto all six walls, a full free-form camera system with mouse-driven rotation/pan/zoom, a 60 FPS game loop, score tracking, and a Game Over screen. Developed as a team project for CS459.

Brady Workplace Safety AI

A mobile app built for the CSI Spring 2025 Hackathon, hosted by UW-Milwaukee in partnership with Brady Corporation. The app uses a fine-tuned Xception CNN to detect OSHA and ANSI regulatory violations from photos in real time. Users can snap a picture or upload an image, and the model instantly classifies it for compliance issues and surfaces a violation report. The ML pipeline was built in Python with TensorFlow/Keras, while the frontend was built with React and TypeScript via Expo Go for cross-platform mobile deployment. A Node.js backend handles server-side logic, and an NLP-based chatbot powered by sentence embeddings assists users in looking up specific regulations.
