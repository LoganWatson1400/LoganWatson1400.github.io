RISA MIPS Assembler

A two-milestone MIPS assembler written in Java that translates Reduced Instruction Set Architecture (RISA) assembly into 8-digit hexadecimal machine code. Milestone 1 handles single-line assembly parsing via a JAR CLI tool; Milestone 2 extends it into a full MIPS simulator capable of processing complete .asm programs with separate .text and .data segments.
